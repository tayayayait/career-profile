솔루션 개요

네 가지 모듈의 목적·주요 기능·기대효과를 카드로 재구성했습니다.

MSDS 조회 (공공데이터포털 API · ChatGPT)

MSDS DB 연동으로 화학물질별 핵심 안전 정보 자동 제공.

  • 유해성·응급조치·화재/누출 대응 자동 요약
  • CAS 등 식별 정보 자동 확인

통합 스마트 안전검색 (산업안전보건공단 API · ChatGPT)

키워드 기반으로 최신 안전·보건 가이드라인 실시간 조회.

  • 사고 유형별 권장 조치 제공
  • 관련 법령·표준 매뉴얼 링크 자동 연결

산업안전보건법 시행규칙 기반 GPTs

시행규칙 자동 해석으로 단계별 사고 대응 절차 구성.

  • NER → 응급조치 → 책임 → 문서 자동화
  • 조항 번호·요약 자동 삽입

회의록 자동 작성 및 배포 (Google API · ChatGPT)

회의 원문을 표준 템플릿으로 자동 요약·정리·배포.

  • 참가자·안건·토의·결론 자동 추출
  • 지정 메일링 리스트로 즉시 전송

Workflow

사고 대응 절차 (시행규칙 기반 GPTs)

사고 유형 인지부터 문서화까지의 자동화 흐름.

STEP 1
사고 유형 분류 (NER)

입력 개요에서 사고 유형 자동 추출·정규화.

STEP 2
초기 응급조치

즉시 필요한 안전·보건 조치 제시.

STEP 3
주체별 책임

사업주·관리감독자·근로자 역할과 책임 정렬.

STEP 4
대응 매뉴얼 & 문서 자동화

관련 조항 번호·요약 삽입, 보고 양식 생성.
